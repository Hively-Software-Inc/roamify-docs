# Introduction

Welcome to Roamify's documentation site. This site is built using [Nextra](https://nextra.vercel.app/), a framework built on top of [Next.js](https://nextjs.org/).

## Coming Soon

This documentation site is still a work in progress. We are working on adding more content to it. If you have any suggestions, feel free to open a pull request or an issue.

## What is Roamify?

Roamify is where you can buy product for your travels. stay connected with your loved ones and share your experiences with the world without worrying about the roaming charges.

## Documentation

The documentation is available at [https://docs.getroamify.com](https://docs.getroamify.com).
