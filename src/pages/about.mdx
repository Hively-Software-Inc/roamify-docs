# About

Roamify eSIM API is a RESTful API that allows you to integrate Roamify services into your application. With Roamify eSIM API, you can buy eSIMs, check your balance, and more.

## Features

- **Buy eSIMs**: You can buy eSIMs using Roamify eSIM API.
- **Check Balance**: You can check your balance using Roamify eSIM API.
- **Top Up**: You can top up your balance using Roamify eSIM API.
- **View Transactions**: You can view your transactions using Roamify eSIM API.
- **Manage eSIMs**: You can manage your eSIMs using Roamify eSIM API.
- **Get eSIM Details**: You can get details of your eSIMs using Roamify eSIM API.
- **Get eSIM Plans**: You can get eSIM plans using Roamify eSIM API.
- **Get eSIM Countries**: You can get eSIM countries using Roamify eSIM API.
- **More**: We are constantly working on adding more features to Roamify eSIM API.
