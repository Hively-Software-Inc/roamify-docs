# Health Check
We provide a health check endpoint that you can use to check the status of Roamify eSIM API. The health check endpoint is available at:

 - Development: [https://api-dev.getroamify.com/api/health](https://api-dev.getroamify.com/api/health)
 - Production: [https://api.getroamify.com/api/health](https://api.getroamify.com/api/health)

You can send a `GET` request to the health check endpoint to check the status of Roamify eSIM API. The response will contain the status of the API.
