# Gift Card Overview
Roamify Gift Card API only works with the Roamify app and currently supports the following features:

- **Create Gift Card**: You can create a gift card using Roamify Gift Card API.
- **Get Gift Card**: You can get a gift card using Roamify Gift Card API.
- **Revoke Gift Card**: You can revoke a gift card using Roamify Gift Card API.
- **More Features**: We are constantly working on adding more features to Roamify Gift Card API.
