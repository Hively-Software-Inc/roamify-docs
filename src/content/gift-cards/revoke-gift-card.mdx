## How to revoke a gift card
To revoke a gift card, you need to send a `POST` request to the `/api/gifts/revoke` endpoint. 

The request body should contain the following:
- `orderId`: The order ID of the gift card. This is a required field of type `string`.

## Usage - revoke a gift card
Here is an example of how you can revoke a gift card using Roamify Gift Card API.

<Tabs items={['curl', 'javascript']}>
    <Tabs.Tab index={0}>
        ```bash
        curl --request POST \
        --url 'https://api-dev.getroamify.com/api/gifts/revoke' \
        --header 'Authorization: Bearer Access Token' \
        --header 'Content-Type: application/json' \
        --data '{
            "orderId": "your-order-id"
        }'
        ```
    </Tabs.Tab>
    <Tabs.Tab index={1}>
        ```javascript
        const axios = require('axios');

        axios.post('https://api-dev.getroamify.com/api/gifts/revoke', {
            orderId: 'your-order-id'
        }, {
            headers: {
                'Authorization': 'Bearer {access token here}',
                'Content-Type': 'application/json'
            }
        })
        .then(response => {
            console.log(response.data);
        })
        .catch(error => {
            console.error(error);
        });
        ```
    </Tabs.Tab>
