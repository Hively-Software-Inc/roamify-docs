## Update Gift Card
To update a gift card, you need to send a `PUT` request to the `/api/gifts/gift` endpoint. 

The request body should contain the following parameters:
- `orderId`: The order ID of the gift card. This is a required field of type `string`.
- `senderEmail`: The new email address of the sender. This is a required field of type `string`.
- `amount`: The new amount of the gift card. This is a required field of type `number`.
- `currency`: The new currency of the gift card. This is a required field of type `string`. The currency should be in [ISO 4217](https://en.wikipedia.org/wiki/ISO_4217) format.
Currently supported currencies are:
    - `USD`: United States Dollar
    - `CAD`: Canadian Dollar
    - `EUR`: Euro
    - `GBP`: British Pound
    - `AUD`: Australian Dollar
    - `CNY`: Chinese Yuan
    - `HKD`: Hong Kong Dollar
    - `JPY`: Japanese Yen
    - `SGD`: Singapore Dollar
    - `TWD`: New Taiwan Dollar
    - `KRW`: South Korean Won
    - `AED`: United Arab Emirates Dirham
    - `MXN`: Mexican Peso
    - `SAR`: Saudi Riyal
    - `EGP`: Egyptian Pound
    - `RUB`: Russian Ruble
    - `QAR`: Qatari Riyal
    - `INR`: Indian Rupee
    - `THB`: Thai Baht
    - `BGN`: Bulgarian Lev
    - `BRL`: Brazilian Real
    - `CHF`: Swiss Franc
    - `CZK`: Czech Koruna
    - `DKK`: Danish Krone
    - `HUF`: Hungarian Forint
    - `MYR`: Malaysian Ringgit
    - `NOK`: Norwegian Krone
    - `NZD`: New Zealand Dollar
    - `PLN`: Polish Zloty
    - `RON`: Romanian Leu
    - `SEK`: Swedish Krona

## Usage - update a gift card
Here is an example of how you can update a gift card using Roamify Gift Card API.

<Tabs items={['curl', 'javascript']}>
    <Tabs.Tab index={0}>
        ```bash
        curl --request PUT \
        --url 'https://api-dev.getroamify.com/api/gifts/gift' \
        --header 'Authorization: Bearer Access Token' \
        --header 'Content-Type: application/json' \
        --data '{
            "orderId": "your-order-id",
            "senderEmail": "test@mail.com",
            "amount": 100,
            "currency": "USD"
        }'
        ```
    </Tabs.Tab>
    <Tabs.Tab index={1}>
        ```javascript
        const axios = require('axios');

        axios.put('https://api-dev.getroamify.com/api/gifts/gift', {
            orderId: 'your-order-id',
            senderEmail: 'test@mail.com',
            amount: 100,
            currency: 'USD'
    }, {
        headers: {
            Authorization: 'Bearer Access Token',
            'Content-Type': 'application/json'
    }
    })
        .then(response => {
        console.log(response.data);
    })
        .catch(error => {
        console.error(error);
    });
        ```
    </Tabs.Tab>
</Tabs>
